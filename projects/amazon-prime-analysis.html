Amazon Prime Subscription Analysis

Project Overview

This project focused on analyzing subscription behavior patterns to identify key retention drivers for Amazon Prime services.

Tools Used

  • Python (Pandas, Scikit-learn, NumPy)
  • XGBoost
  • Random Forest

Skills Applied

  • Ensemble Modeling
  • Feature Engineering
  • Predictive Analytics

Methodology

The analysis followed these key steps:

  1. Data collection and preprocessing
  2. Exploratory data analysis to identify patterns
  3. Feature engineering to create relevant variables
  4. Model development using ensemble techniques
  5. Evaluation and interpretation of results
